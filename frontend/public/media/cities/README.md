# City Media Placeholders

Place licensed, optimized cinematic city media in this directory:

- `rome.mp4` and `rome.jpg`
- `barcelona.mp4` and `barcelona.jpg`
- `berlin.mp4` and `berlin.jpg`
- `copenhagen.mp4` and `copenhagen.jpg`
- `bari.mp4` and `bari.jpg`
- `global.mp4` and `global.jpg`

The city background loads local MP4 footage first on motion-capable devices, uses the matching JPG for reduced-motion or when video is unavailable, and displays the atmospheric gradient when no file exists.

Keep footage short, compressed, loop-friendly and cleared for product use. The UI remains functional when any or all media slots are absent.
